Chat Input — Slash Command Style

How should slash commands appear when user types "/"?

📝
/quiz
Generate quiz on current topic
📚
/resources
Find learning resources
📊
/plan
Create or view study plan
📎
/pdf
Chat with uploaded PDF
/

A: Popup Command Palette

Type "/" and a floating menu appears above input. Filter by typing. Select with arrow keys or click. Disappears on escape.

B: Inline Chips

Chip buttons appear above input when "/" typed. Always visible as row of pills. Clicking inserts the command. Filters as you type.
